📝 How to Add Your Projects

  1. Open projects.html in your code editor
  2. Find the <!-- Add your projects here --> comment below
  3. Copy the example project card structure
  4. Replace the details with your project information:
    • Icon: Change the emoji (🚀, 💼, 📊, 🎨, etc.)
    • Title: Your project name
    • Description: Brief explanation of your project
    • Tech badges: Technologies used
    • Link: URL to your live project or GitHub repo
  5. Save the file and refresh your browser to see changes
